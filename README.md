# Power BI Learning Journey 📊

A structured, hands-on repository documenting my complete learning path in **Microsoft Power BI** — from the fundamentals of the interface and data transformation to advanced DAX, data modelling, and interactive report design.

Each folder represents a topic covered during my learning, containing practice files, `.pbix` reports, and notes.

## 📁 Repository Structure

| # | Topic | Description |
|---|-------|-------------|
| 0 | **Power BI Introduction** | Overview of Power BI, its components (Desktop, Service, Mobile), and the interface. |
| 1 | **Data Transformation** | Basics of Power Query — shaping and cleaning raw data. |
| 2 | **Numeric, Date & Time Conditional** | Conditional logic and transformations on numeric, date, and time columns. |
| 3 | **Column Formatting, Merge & Append Queries** | Formatting columns and combining datasets using Merge and Append Queries. |
| 4 | **Creating Tables, Pivoting & Unpivoting** | Reshaping data through table creation, pivoting, and unpivoting techniques. |
| 5 | **Assignment** | Practice assignment applying data transformation concepts. |
| 6 | **Data Modelling** | Fundamentals of building relationships between tables. |
| 7 | **Practice Data Modelling Set** | Hands-on exercises to reinforce data modelling concepts. |
| 8 | **School Data Modelling** | A real-world style case study modelling a school dataset. |
| 9 | **Calculated Columns** | Creating and using calculated columns in DAX. |
| 10 | **Measures and Its Types** | Introduction to DAX measures and their categories. |
| 11 | **Types of DAX Functions** | Overview of the different categories of DAX functions. |
| 12 | **Power BI DAX Logical Functions** | Logical functions such as IF, AND, OR, SWITCH, etc. |
| 13 | **Power BI DAX Operators** | Arithmetic, comparison, logical, and text operators in DAX. |
| 14 | **Power BI Visuals and Filter Options** | Working with core visuals and filter panes/options. |
| 15 | **Matrix, Slicers, Numbers and Text Cards** | Building matrices, slicers, and KPI-style number/text cards. |
| 16 | **KPI Visuals, Maps, and Tree Visuals** | KPI indicators, map visuals, and tree/hierarchy visuals. |
| 17 | **Tooltips, Colors and Themes, Bookmarks** | Enhancing reports with custom tooltips, themes, and bookmarks. |
| 18 | **Phone and Desktop Report View** | Designing reports optimized for both desktop and mobile views. |

## 🎯 Purpose

This repository serves as:
- A personal learning log and reference guide for Power BI concepts.
- A portfolio showcasing progressive, hands-on practice — from data transformation to DAX and visualization.
- A resource for anyone starting their Power BI learning journey, structured topic-by-topic.

## 🛠️ Tools Used

- **Power BI Desktop**
- **Power Query Editor**
- **DAX (Data Analysis Expressions)**

## 📌 How to Use

1. Clone or download this repository.
2. Navigate to the topic folder you're interested in.
3. Open the `.pbix` file(s) in Power BI Desktop to explore the reports and models.

```bash
git clone <repository-url>
```
